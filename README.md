## 使用说明
- replit环境选择`Bash`
- 不建议使用免费版部署，会泄漏数据库源文件

```bash
bash <(curl -s https://raw.githubusercontent.com/yzsong06/Replit-Flarum/main/install.sh)
```
- 执行上面脚本
- 最后点击Run运行项目即可。

## 数据库设置
- 数据库名称：MySql
- 数据库用户名：root
- 数据库密码：root
